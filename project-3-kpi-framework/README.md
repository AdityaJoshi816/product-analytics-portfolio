# Project 3: KPI Framework — Leading Retail Investment Platform

## Overview
Complete metrics framework for a retail investment platform
defining North Star Metric, L1 and L2 metrics.

## North Star Metric
Monthly Active Investors (MAI) — unique users completing
at least one investment transaction per month

## L1 Metrics
- Acquisition: Qualified New Investor Rate
- Activation: First Investment Conversion Rate
- Retention: Monthly Investment Retention Rate

## L2 Metrics
- App Store Conversion Rate
- Referral Activation Rate
- KYC Drop-off Rate
- Time to First Investment
- SIP Continuity Rate
- Portfolio Engagement Rate

## Key Insights
1. KYC completion is the highest-leverage drop-off point
2. First 7 days post-KYC are critical for first investment
3. SIP continuity compounds MAI growth month over month
