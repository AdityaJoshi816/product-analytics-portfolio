### 2. A/B Test Case Study — Q-Commerce Checkout Funnel
Designed and analyzed a two-proportion z-test on a quick-commerce checkout flow experiment, applying statistical rigor to validate significance and translate the lift into projected business impact.

**Key findings**
- Conversion lift: **+26.1%** (p = 0.000037)
- Projected monthly revenue impact: **₹16.7 Lakh**
- Validated sample size, power, and guardrail metrics before declaring significance

**Stack:** Python (SciPy, Pandas), Statistical Hypothesis Testing
